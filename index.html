Welcome to the Crunching Code Framework

The Crunching Code Framework is a rapid application development built on Java which aims to help you build business applicaitons much quicker by tackling common problems and tasks which you might encounter and abstracting them into a framework that you can use as a base of your applicaiton or extend and use as you see fit. You can help contribute to this project by pulling the latest code and exploring the framework. Check out the new branch:

$ cd your_repo_root/repo_name
$ git fetch origin
$ git checkout gh-pages

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.

Crunching Code Components

We have tackled some very common and sometimes not so common tasks and problems in the framework to make building applications much easier for you. Some of our modules include:

  • SMS
  • USSD
  • Business Model
  • Authentication
  • Social Integration
  • Reporting
  • Templates (Mobile Applications & Web Applications)
  • much more...
